# Quiz Master (ويب للاختبارات الرقمية)

واجهة ويب خفيفة لإدارة اختبارات اختيار من متعدد مع دعم لغتين (عربي/إنجليزي)، ثيمين (فاتح/داكن)، وتحليلات محلية وصندوق مراجعة.

## المزايا الرئيسية
- **تحميل أسئلة من JSON** مع دعم `group`, `sub_group`, مسارات صور محلية أو روابط خارجية، وخيارات افتراضية `A/B/C/D` يمكن تخصيصها عبر `choices`.
- **اختيار أقسام ومجموعات فرعية** قبل بدء الاختبار.
- **وضع ذكي** يختار أسئلة غير محلولة أو تكررت فيها الأخطاء وفق السجل المحلي.
- **صندوق مراجعة** لتجميع الأسئلة الموسومة أو الخاطئة، مع سجل دائم في المتصفح.
- **التحليلات**: دقة إجمالية وإحصائيات لكل مجموعة.
- **تبديل اللغة والثيم** مع حفظ التفضيلات في `localStorage`.
- **تهيئة المزامنة** من خلال ملف `data/sync_config.json` وزر تشغيل/إيقاف تحضيرًا لربطها بباك-إند PHP بسيط (موجود في مجلد `backend/`).
- **تحميل مسبق للصور** عند بدء الاختبار لتقليل زمن الانتظار بين الأسئلة.

## هيكلة المشروع
```
index.html            # الواجهة الرئيسية باستخدام Bootstrap 5
assets/css/style.css  # تحسينات شكلية بسيطة ودعم RTL/LTR
assets/js/app.js      # منطق الواجهة وإدارة الأسئلة والسجل
data/questions.json   # مصدر الأسئلة بصيغة JSON
data/sync_config.json # إعداد أولي لتفعيل/تعطيل المزامنة
backend/index.php     # نقطة بداية مستقبلية للـ API (Placeholder)
```

## تنسيق ملف الأسئلة
مثال:
```json
[
  {
    "path": "https://example.com/01.png",
    "group": "رياضيات",
    "sub_group": "جبر",
    "correct": "A",
    "choices": ["5", "6", "7", "8"]
  }
]
```
- `choices` اختيارية، وعند غيابها تستخدم القيم الافتراضية `A/B/C/D`.
- `path` يمكن أن يكون رابطًا خارجيًا أو مسارًا محليًا.

## التشغيل محليًا
1. خدم الملفات استاتيكيًا (مثال: `php -S 0.0.0.0:8000`).
2. افتح المتصفح على `http://localhost:8000`.
3. عدّل `data/questions.json` لإضافة أسئلتك.

## المزامنة المستقبلية
- يمكن تفعيل/إيقاف صلاحية المزامنة من البطاقة داخل التطبيق. الإعداد الافتراضي يقرأ من `data/sync_config.json`.
- ملف `backend/index.php` موجود كبداية بسيطة لتوصيل API لاحقًا (غير مستخدم حاليًا).

## الملاحظات
- كل البيانات (السجل، التفضيلات، صندوق المراجعة) مخزنة في `localStorage` لحين ربط المزامنة.
- يدعم RTL/LTR تلقائيًا حسب اللغة المختارة.
